---
title: Event Summary
---

Memobase will summarize the recent chats into a concise summary, which will guide the following profile/event tag extraction.
The coverage and quality of this summary is cruical to the memories.

Memobase allows you to customize the summary prompt to further tune the summary.

## Customize the summary prompt
You can customize the summary prompt in your `config.yaml`, for example:
```yaml config.yaml
event_theme_requirement: "Focus on the user's infos, not its instructions."
```
